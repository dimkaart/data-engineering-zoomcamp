dbt:
    - run all models: dbt run
    - run all models including seed and tests: dbt build
        --> Example: dbt build
        --> build only one model: dbt build --select $MODEL_NAME (Example: dbt build --select fact_trips)
        --> build only one model AND everything that it needs: dbt build --select +$MODEL_NAME (Example: dbt build --select +fact_trips)
    - install all dependencies: dbt deps
    - set up variables: dbt run --var '$VAR_NAME:$VAR_VALUE'
        --> Example: dbt run --var 'is_test_run: false'
    - upload csv to BQ: dbt speed   
        --> save csv in folder seeds and run 'dbt seed'
        --> update table manually: dbt run --full-refresh (drop and create new table with same name)
